## 🔧 작업 방식 메모 (중요)

- **수정 → 올리기 흐름**: index.html 등을 수정한 뒤 git add/commit/push 하면 1~2분 뒤 https://hpos.co.kr/ 에 반영됨
- **git 명령 실행 시 주의**: Git이 세션 중간에 설치되어, PowerShell에서 git 쓰기 전 PATH 새로고침 필요:
  `$env:Path = [System.Environment]::GetEnvironmentVariable("Path","Machine") + ";" + [System.Environment]::GetEnvironmentVariable("Path","User")`
- **두 컴퓨터 동기화 규칙**:
  - 작업 시작 전: `git pull` ("최신으로 가져와줘")
  - 작업 끝난 후: `git push` ("인터넷에 올려줘")
- **새 컴퓨터 세팅 순서**: VS Code 설치 → Claude Code 확장 설치·로그인(x26589334@gmail.com) → Git 설치 → `git clone https://github.com/x26589334-cpu/yoon.git`
